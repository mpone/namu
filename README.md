## About
The landing page, created for National Art Museum of Ukraine.

Landing page was developed with:

- HTML5
- CSS3
- SCSS
- BEM
- JavaScript
- Figma

The project has the responsive layout for several screen-widths:
- Large screens 2560px
- Full HD 1920px
- The design 1600px
- Notebook 1280px
- Tablet 1024
- Mobile (> 320px)
